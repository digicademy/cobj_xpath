Installation
------------

Requirements
^^^^^^^^^^^^

- You need at least PHP 5.3+

- You need libxml and SimpleXML enabled

- The extension has been tested on TYPO3 versions 6.2 to 7.x

Installation
^^^^^^^^^^^^

Simply import it from TER and install. Nothing else needs to be done.
